# Open Policy Agent (OPA) Gemini Context

## Focus Area
**Policy as Code**

## Why Index?
Helps standardize the logic in your `PolicyEngine` against enterprise norms.

## Project Description
General-purpose policy engine that enables unified, context-aware policy enforcement across the entire stack using the Rego language.

## Key Concepts
- **Rego**: The declarative query language used for policies.
- **Policy Decoupling**: Separating policy decision from enforcement.
- **Data + Policy**: OPA evaluates inputs against policy and data.

## Conventions
- **Language**: Go (System), Rego (Policy).
- **Style**: Declarative policies.
